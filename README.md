# Issue Tracker — Frontend

React frontend for the Issue Tracker helpdesk system.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS v3
- shadcn/ui
- TanStack Query v5
- Zustand
- React Router v6
- Vitest + React Testing Library

## Development

```bash
npm install
npm run dev
```

## Testing

```bash
npm run test
```

## Build

```bash
npm run build
```
